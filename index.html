About Me

Hi, I'm a recent graduate from Universiti Teknologi Malaysia (UTM), holding a Bachelor's degree in Computer Science (Graphics and Multimedia Software). I'm passionate about both playing and developing games, with a strong interest in creating interactive, meaningful, and visually engaging experiences.

During my time at university, I studied a wide range of game design concepts—from narrative structure and level design to UI/UX and player engagement theory. I've applied this knowledge through hands-on development of several game projects, both individually and as part of a team. These projects allowed me to explore gameplay mechanics, and player experience from prototype to polish.

I'm deeply interested in the intersection of creativity and technology. Whether it's crafting gameplay systems, designing intuitive interfaces, or optimizing performance for smoother play, I strive to bring thoughtful design and technical precision together.

Escape the Room

GDevelop | 2D Puzzle Adventure

  • Designed and developed a 2D puzzle-adventure game where the player must escape a locked room.
  • Implemented core mechanics:
    • Movement (WASD + ladder climbing + jumping).
    • Puzzle solving (find password, unlock box, locate hidden key).
    • Narrative system with NPC dialogue and branching choices.
  • Created room layout with walls, ladders, furniture, and NPC using GDevelop behaviors.
  • Integrated interactive puzzle system using variables, collision detection, and text input logic.
  • Debugged and refined collision and input validation to ensure smooth gameplay.

Coastal Exploration

Babylon.js | HTML + JavaScript

  • Designed and developed a 3D coastal exploration game set on a tropical island with interactive NPCs and collectibles.
  • Implemented gamification elements:
    • Exploration: players discover items and NPCs across the island.
    • Scoring system: coins earned by collecting seashells/treasures and selling them to a pirate.
    • Quests & Rewards: complete tasks for extra coins, which unlock hidden objects (pirate ship, beach chairs, etc.).
  • Integrated 3D assets using BABYLON.SceneLoader.ImportMeshAsync and customized positioning, scaling, and animations.
  • Created environment effects: skybox, ocean shader (WaterMaterial), lighting, and smooth WASD + camera movement.
  • Built interaction systems for collecting items, NPC dialog, and unlockable features; quest system tracked progress and rewarded players dynamically.
  • Deployed on GitHub Pages for public access.
